55 /*! \brief This implements line plots where the data is taken from a user supplied function \f$ y=f(x) \f$
56  \ingroup jkqtplotter_functiongraphs
57 
58  This class implements an intelligent plotting algorithm for functions. It starts by sampling
59  the function at minSamples positions. Then each function interval is bisected recursively if
60  necessary. To do so the function is evaluated at the mid point and the slopes \f$ \alpha_{\mbox{left}} \f$
61  and \f$ \alpha_{\mbox{right}} \f$ of the two linear segments are compared. the midpoint is added
62  to the graph if \f[ \left|\alpha_{\mbox{right}}-\alpha_{\mbox{left}}\right|>\mbox{slopeTolerance} \f]
63  In addition all sampling points except minimum and maximum are beeing shifted by a random fraction their
64  distance to the other points. This helps to prevent beats when sampling periodic functions.
65 
66  the following image
67  \image html plot_functionplots.png
68 
69  \see \ref JKQTPlotterFunctionPlots, jkqtpstatAddPolyFit(), jkqtpstatAddWeightedRegression(), jkqtpstatAddRobustIRLSRegression(), jkqtpstatAddRegression(), jkqtpstatAddLinearWeightedRegression(), jkqtpstatAddRobustIRLSLinearRegression(), jkqtpstatAddLinearRegression()
70  */
