JKQTPXFunctionLineGraph Class Reference
JKQTPlotter: Qt based Scientific Plotter Class » Plot Elements » Graph Classes » Line/Symbol Graphs » Function Graphs

This implements line plots where the data is taken from a user supplied function This class implements an intelligent plotting algorithm for functions. It starts by sampling the function at minSamples positions. Then each function interval is bisected recursively if necessary. To do so the function is evaluated at the mid point and the slopes and of the two linear segments are compared. the midpoint is added to the graph if

In addition all sampling points except minimum and maximum are beeing shifted by a random fraction their distance to the other points. This helps to prevent beats when sampling periodic functions. More...

Detailed Description

This implements line plots where the data is taken from a user supplied function

This class implements an intelligent plotting algorithm for functions. It starts by sampling the function at minSamples positions. Then each function interval is bisected recursively if necessary. To do so the function is evaluated at the mid point and the slopes and of the two linear segments are compared. the midpoint is added to the graph if

In addition all sampling points except minimum and maximum are beeing shifted by a random fraction their distance to the other points. This helps to prevent beats when sampling periodic functions.
